# Think Customer Support Starter

A [Think](https://www.npmjs.com/package/@cloudflare/think) customer support agent
on Cloudflare Workers, with a small React chat UI.

It adds a custom `lookupOrder` tool alongside the built-in workspace tools and a
colocated `escalation` skill for handing off to a human.

## Develop

```sh
npm install
npm run dev
```

Open the printed URL and start chatting.

## What's here

- `agents/support/agent.ts` — the agent: `getTools()` for custom tools plus a
  skill runner.
- `agents/support/skills/` — colocated skills bundled via `agents:skills`.
- `src/client.tsx` — a React chat client using `useAgent` + `useAgentChat`.
- `wrangler.jsonc` / `think.d.ts` — generated by the Think framework. Re-run
  `npm run types` after changing agents or bindings.

## Deploy

```sh
npm run deploy
```
