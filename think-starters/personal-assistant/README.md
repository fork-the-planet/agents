# Think Personal Assistant Starter

A [Think](https://www.npmjs.com/package/@cloudflare/think) personal assistant on
Cloudflare Workers, with a small React chat UI.

It has durable memory (a writable `memory` context block the model updates on
its own and that survives restarts) and a proactive scheduled task that runs
even when nobody is connected.

## Develop

```sh
npm install
npm run dev
```

Open the printed URL and start chatting.

## What's here

- `agents/assistant/agent.ts` — the agent: `configureSession()` for memory and
  `getScheduledTasks()` for proactive runs.
- `src/client.tsx` — a React chat client using `useAgent` + `useAgentChat`.
- `wrangler.jsonc` / `think.d.ts` — generated by the Think framework. Re-run
  `npm run types` after changing agents or bindings.

## Deploy

```sh
npm run deploy
```
